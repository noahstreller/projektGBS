Aufgabe 5

Phase Praktisches Beispiel zur Phase Erkenntnisse / Erfahrungen
Informieren Mit dem Auftragsgeber die Ziele besprechen Ein einzelnes Gespräch mit dem Auftragsgeber wird nie reichen.
Planen Die Arbeitspakete erstellen und verteilen Das Planen sollte nicht zu kurz kommen.
Entscheiden Entscheidung zwischen verschiedener Frameworks Benötigt in der Regel nicht allzuviel Zeit.
Realisieren Das Programm entwickeln Dies wird die meiste Zeit in Anspruch nehmen. Man sollte auf eine gute Übersicht achten.
Kontrollieren Das Programm testen Falls beim Kontrollieren ein Fehler gefunden wird, kommt wieder das Realisieren an die Reihe.
Auswerten Sich über das Projekt, die Ergebnisse und die Arbeit Gedanken machen Den Code eines alten Projekts NIE löschen, denn man wird es eines Tages brauchen.
